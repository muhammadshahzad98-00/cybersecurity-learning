### 3. Careers in Cyber (Completed: 7 April 2026)

**What I learned:**
- Cybersecurity has multiple roles (offensive, defensive, and others)
- Red team focuses on attacking systems ethically
- Blue team focuses on defending and monitoring systems

**Key roles:**
- Penetration Tester (offensive)
- SOC Analyst (defensive)
- Security Engineer

**My understanding:**
Cybersecurity offers different career paths, and beginners often start in defensive roles like SOC Analyst.
